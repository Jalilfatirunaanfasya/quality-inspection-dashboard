# quality-inspection-dashboard
This project analyzes anonymized manufacturing quality inspection data to identify defect trends, root causes, machine performance, and process improvement opportunities.

## Project Objectives
- Analyze defect frequency
- Identify recurring root causes
- Compare machine-related quality issues
- Evaluate shift performance
- Support data-driven manufacturing decisions

## Tools Used
- Python
- Pandas
- Matplotlib / Plotly
- Google Colab / Jupyter Notebook

## Dataset
Anonymized manufacturing inspection data from Oct 2024 – Jan 2026.

- Interactive dashboards
